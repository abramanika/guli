Button is Guli's pill-shaped action control; saperavi fill means "primary action" everywhere.

```jsx
<Button variant="primary">გაგრძელება</Button>
<Button variant="ghost" size="md">გაზიარება</Button>
<Button loading>ᲓᲐᲘᲬᲧᲔ</Button>
```

Rules: never truncate Georgian labels (width grows); Mtavruli (ᲓᲐᲘᲬᲧᲔ) only on ≤2-word labels; destructive only for data deletion.
