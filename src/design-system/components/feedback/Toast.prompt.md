Toast (pill notice) and Skeleton (shimmering loading shape) — Guli's quiet system-state components.

```jsx
<Toast floating>კავშირი გაწყდა — პასუხები შენახულია</Toast>
<Skeleton width={220} height={16} />
```

Copy rule: plain statement + fix, never apologetic. Diffusion animation is never used as a loader.
