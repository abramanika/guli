TraitBar shows one Big Five dimension as a band between two Georgian pole labels (e.g. გულჩათხრობილი ↔ გულღია) with a droplet marker.

```jsx
<TraitBar trait="E" value={72} percentileLabel="72%" />
<TraitBar trait="C" value={64} secondValue={31} />  // pair comparison: droplet vs ring
```

Pole labels come from the canonical TRAITS map automatically. Markers differ by shape (droplet vs ring) so pairs are distinguishable without color.
